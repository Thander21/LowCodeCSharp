# LowCodeCSharp

Esta é uma extensão do VS Code para gerar diagramas de fluxo de código C#.

## Funcionalidades

- Analisa projetos C# e gera diagramas de fluxo.

## Instalação

Para instalar a extensão, execute `npm run package` e siga as instruções.
